# Сборка Censr в установщик

Сборка делается **на Windows**. Linux/macOS не подходят — PyInstaller собирает
под ту ОС, где запущен, а установщик делается Inno Setup (только Windows).

## Что понадобится

- **Python 3.10+** в PATH (проверь: `py -3 --version`)
- **Интернет** — при первой сборке скачается ffmpeg (~80 МБ)
- **Inno Setup 6** (необязательно, для .exe-установщика):
  https://jrsoftware.org/isdl.php
  Без него получится «портативная» папка, которую можно просто заархивировать.

## Быстрый старт

Из корня проекта:

```bat
packaging\build.bat
```

Скрипт сам по шагам:

1. создаёт виртуальное окружение `.venv`;
2. ставит зависимости из `requirements.txt` + PyInstaller;
3. скачивает ffmpeg (если его ещё нет в `packaging\ffmpeg\`);
4. собирает два exe в `dist\Censr\`:
   - **Censr.exe** — графическая версия (двойной клик, без консоли);
   - **Censr-cli.exe** — консольная, для скриптов и пакетной обработки;
5. кладёт рядом папки `models\` (модель распознавания) и `ffmpeg\`;
6. если установлен Inno Setup — собирает установщик в `dist\installer\`.

## Результат

- **Портативная версия:** папка `dist\Censr\` целиком. Заархивируй и передавай —
  у пользователя ничего ставить не нужно, всё внутри (Python, ffmpeg, модель).
- **Установщик:** `dist\installer\Censr-Setup-1.4.0.exe` — обычная установка
  с ярлыками в меню «Пуск» и (по желанию) на рабочем столе. Прав администратора
  не требует (ставится в профиль пользователя).

## ffmpeg вручную (если скачивание не сработало)

Положи `ffmpeg.exe` и `ffprobe.exe` в `packaging\ffmpeg\` и запусти `build.bat`
снова — шаг скачивания будет пропущен. Сборки: https://www.gyan.dev/ffmpeg/builds/

## Антивирусы и ложные срабатывания

Неподписанные exe от PyInstaller иногда триггерят ML-эвристики антивирусов
(Wacatac, Trojan:Win32/Wacatac.B!ml и т. п.) — это **ложные срабатывания**, не
вирус. Уже сделано для снижения:

- заполнены метаданные версии (`version_info.txt`) — издатель, описание, версия;
- отключён UPX-сжиматель (частый триггер).

Чтобы убрать почти полностью — **подписать exe** сертификатом Authenticode
(code signing). Без подписи можно отправить файл как ложное срабатывание в
Microsoft: https://www.microsoft.com/wdsi/filesubmission

## Поднять версию

Поменяй число в трёх местах: `version_info.txt` (`filevers`/`FileVersion`/
`ProductVersion`) и `censr.iss` (`#define MyAppVersion`).

## Состав папки packaging

| Файл | Назначение |
|---|---|
| `build.bat` | главный сборочный скрипт (ASCII, без кириллицы) |
| `censr.spec` | конфигурация PyInstaller (два exe, сбор зависимостей) |
| `launch_gui.py` / `launch_cli.py` | точки входа сборок |
| `version_info.txt` | метаданные версии Windows |
| `censr.iss` | скрипт установщика Inno Setup |
| `ffmpeg\` | сюда кладётся ffmpeg (создаётся скриптом) |
| `dist-extra\` | необязательно: файлы, копируемые в готовую папку как есть |

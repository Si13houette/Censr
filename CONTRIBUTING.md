# Участие в разработке

Спасибо за интерес к Censr! Любой вклад приветствуется — от опечатки в README
до новой фичи.

## Сообщить о баге или предложить идею

Открой [issue](https://github.com/Si13houette/Censr/issues). Для бага полезно приложить:

- что делал и что ожидал увидеть;
- что произошло вместо этого;
- версию Censr и Windows;
- содержимое `censr_error.log` (лежит рядом с `Censr.exe`), если приложение упало.

## Запуск из исходников

```bat
pip install -r requirements.txt
python -m censr              :: GUI
python -m censr file.mp3     :: CLI
```

Нужен ffmpeg в PATH. При первом запуске модель (~250 МБ) скачивается с Hugging Face
автоматически (см. [README](README.md#-запуск-из-исходников)).

## Тесты

```bat
pip install pytest
pytest
```

Перед пул-реквестом убедись, что тесты в `tests/` проходят. Новую логику —
по возможности покрывай тестом.

## Стиль и принципы

- **Простота важнее изящества ради изящества.** Минимальное воздействие на код,
  без лишних абстракций.
- Версия проекта живёт в одном месте — `censr/__init__.py` (`__version__`).
  Меняй только её.
- Тонкая геометрия зон глушения — константы в `censr/audio_zone.py`.

## Пул-реквесты

1. Форкни репозиторий и создай ветку от `main`.
2. Сделай изменения отдельными осмысленными коммитами.
3. Проверь, что тесты проходят.
4. Открой PR с кратким описанием: что и зачем.

# Сборка Censr (Windows)

Один батник собирает портативную папку, zip и установщик.

## Что нужно один раз поставить
- **Python 3.10+** (x64), отмечен «Add to PATH».
- **Inno Setup 6** — для установщика: https://jrsoftware.org/isdl.php
  (без него соберётся только портатив — это не ошибка).
- **ffmpeg.exe + ffprobe.exe** — положи их в папку `ffmpeg\` в корне проекта.
  Если их там нет, батник возьмёт из системного PATH; если и там нет —
  соберёт без них (тогда ffmpeg должен быть в PATH у пользователя).

Зависимости Python (`requirements.txt`) и PyInstaller батник ставит сам.

## Запуск
```
build.bat
```

## Что получится
- `dist\Censr\Censr.exe` — портативная сборка (папка целиком переносимая).
- `installer\Output\Censr-portable-<версия>.zip` — та же папка архивом.
- `installer\Output\Censr-Setup-<версия>.exe` — установщик (ярлыки, аптейт, удаление).

Версия берётся из `censr/__init__.py` (`__version__`). Чтобы поднять версию —
правь только её.

## Состав сборки
- `run_censr.py` — точка входа (эквивалент `python -m censr`).
- Модель `models\gigaam-v3-onnx\` (215 МБ) кладётся рядом с exe → работает офлайн.
- `censr.ico` и `ffmpeg\` — рядом с exe (так их находит приложение).
- Установка по умолчанию в профиль пользователя (`%LocalAppData%\Programs\Censr`),
  без прав администратора.

## Если что-то не так
- **Нет звука в «проверке/прослушать»** — добавь в команду PyInstaller в `build.bat`
  флаг `--collect-all PySide6` (вшивает все Qt-плагины мультимедиа; сборка тяжелее).
- **Падает на старте** — смотри `censr_error.log` рядом с `Censr.exe`.
- **«ModuleNotFoundError» при запуске** — добавь недостающий пакет флагом
  `--collect-all <имя>` (или `--hidden-import <имя>`) в `build.bat`.
